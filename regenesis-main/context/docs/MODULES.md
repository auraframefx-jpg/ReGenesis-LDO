## ğŸ—ï¸ Architecture Highlights

### Build System
- **Convention Plugins**: `genesis.android.application` and `genesis.android.library` provide consistent configuration
- **Gradle Version Catalog**: Centralized dependency management in `libs.versions.toml`
- **KSP**: Kotlin Symbol Processing for annotation processing
- **Compose**: Modern declarative UI throughout

### Technology Stack
- **Language**: Kotlin 2.3.0-Beta2
- **UI**: Jetpack Compose with Material Design 3
- **DI**: Hilt for dependency injection
- **Async**: Coroutines and Flow
- **Storage**: Room, DataStore, encrypted SharedPreferences
- **Network**: Retrofit, OkHttp, WebSockets
- **Root**: LibSU for system-level access
- **Framework**: LSPosed/Xposed via YukiHook API

### Key Design Patterns
- **MVVM**: Model-View-ViewModel for UI layers
- **Repository Pattern**: Data access abstraction
- **Dependency Injection**: Hilt-powered DI throughout
- **Reactive Streams**: Flow-based reactive programming
- **Clean Architecture**: Clear separation of concerns
