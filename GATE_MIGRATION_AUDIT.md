## NEW DESIGN SYSTEM REQUIREMENTS

### **Core Visual Elements**
1. **Triple-Layer Neon Border System**
   - Outer glow (soft, wide radius)
   - Middle border (sharp cyan line)
   - Inner accent (corner highlights)
   - All with pulsing animation

2. **Perspective Tilt Effect**
   - 3D rotation on Y-axis
   - Depth shadow beneath card
   - Parallax scrolling effect

3. **Icon System**
   - Row of 4-6 icons at top of card
   - Geometric vector style (not pixel art)
   - Color-coded (cyan, orange, purple accents)
   - Floating/hovering effect

4. **Background Elements**
   - Dark blue to black gradient
   - Scanline overlay
   - Matrix-style code rain (optional)
   - Circuit board patterns
   - Particle effects

5. **Typography**
   - Large, bold title in center
   - Futuristic sans-serif font
   - Letter spacing increased
   - Glowing outline effect
   - Optional: Striped/diagonal line pattern through letters

6. **Content Area**
   - Pixel art characters/scenes (maintained but enhanced)
   - OR code editor view with typing hands
   - OR geometric icon compositions
   - Holographic frame around content

7. **Footer Button**
   - "MENU THEMES" or action button
   - Rounded rectangle with border
   - Glowing on hover
   - Consistent placement at bottom

### **Animation System**
- Pulse breathing effect (border glow)
- Floating/hovering animation
- Scanline sweep
- Particle drift
- Icon rotation/bounce on interaction
- Card tilt on scroll

### **Updated GateConfig Data Class**
```kotlin
data class GateConfig(
    // Existing fields...
    val moduleId: String,
    val title: String,
    val route: String,

    // New fields for tech noir style
    val primaryIcon: @Composable () -> Unit,  // Main geometric icon
    val topIcons: List<@Composable () -> Unit> = emptyList(),  // Icon row
    val accentColor: Color,  // Unique accent (orange, purple, pink, etc.)
    val contentComposition: ContentType,  // PIXEL_ART, CODE_VIEW, GEOMETRIC
    val customContent: (@Composable () -> Unit)? = null,  // Custom center content
    val buttonText: String = "MENU THEMES",  // Footer button
    val enableParticles: Boolean = true,
    val perspectiveDegree: Float = 5f,  // Tilt angle
)

enum class ContentType {
    PIXEL_ART,      // Show existing pixel art with holographic frame
    CODE_VIEW,      // Code editor view (Code Assist)
    GEOMETRIC,      // Large geometric icon composition
    CUSTOM          // Use customContent composable
}
```
